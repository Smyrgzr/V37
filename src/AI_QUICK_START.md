## ðŸ†˜ Troubleshooting

### "AI insights not showing"
- Check if AI features are enabled (Settings)
- Ensure you have sufficient data (min 30 days)
- Refresh browser cache
- Contact support if issue persists

### "Churn predictions seem inaccurate"
- AI needs time to learn (2-3 weeks)
- Verify customer data is up-to-date
- Check if recent promotions affected behavior
- Models retrain weekly for accuracy

### "Dynamic pricing too aggressive"
- Reduce max price increase in Settings
- Disable specific rules (e.g., weather)
- Set manual approval for large changes
- Review competitor pricing benchmarks

### "Damage detection misses scratches"
- Ensure good lighting in photos
- Take photos from multiple angles
- Clean vehicle surface before scanning
- Minimum photo resolution: 1080p
